Si el valor del objeto va a cambiar un gran numero de veces pero solo sera editado por un unico hilo entonces es recomendable utilizar
com.utilities.text.StringBuilder

- Velocidad de ejecución : 297
- USO DE CONTRUCTORES DENTRO DE STRINGBUILDER

com.utilities.text.StringBuilder()

Construye un com.utilities.text.StringBuilder vacío y con una capacidad por defecto de 16 carácteres.

com.utilities.text.StringBuilder s = new com.utilities.text.StringBuilder();

com.utilities.text.StringBuilder(int capacidad)

Se le pasa la capacidad (número de caracteres) como argumento.

com.utilities.text.StringBuilder s = new com.utilities.text.StringBuilder(55);

com.utilities.text.StringBuilder(String str)

Construye un com.utilities.text.StringBuilder en base al String que se le pasa como argumento.

com.utilities.text.StringBuilder s = new com.utilities.text.StringBuilder("hola");
